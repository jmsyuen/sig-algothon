#  STRATEGY: Z-score mean reversion
For each asset, look at its average price over the last lookback days (we can play with the *lookback* variable to see how results change). If today's price is unusually high above that average, we bet it falls back (go short). If it's unusually low, we bet it rises back (go long). "Unusually" is measured with a z-score: how many standard deviations away from the recent average today's price is.
